Подробнее об анализаторе

Olympus Vanta Element - высокоточный анализатор драгоценных металлов. Благодаря технологии рентгенофлуоресцентного анализа Vanta Element за считанные секунды выполняет точный элементный анализ любого изделия. С помощью анализатора мы можем определить чистоту и пробу изделия, не повреждая его целостность.

 

Анализатор легко определяет точное содержание драгоценных металлов - ЗОЛОТА, ПЛАТИНЫ, ПАЛЛАДИЯ, РОДИЯ, СЕРЕБРА. Кроме того Vanta Element с точностью выявит содержание широкого спектра элементов: Cd, Ti, V, Cr, Mn, Fe, Co, Ni, Cu, Zn, W, Hf, Ta, Re, Pb, Bi, Zr, Nb, Mo, Pd, Ag, Sn, Sb.

Мы предлагаем услугу по оценке любых изделий из драгоценных металлов с выдачей соответствующего сертификата.
